# Browser-History-Simulator
A Java project that simulates browser history using Stack Data Structure.

## 🚀 Features
- Visit a new page
- Go back to the previous page
- View the current page
- Show complete browser history

## 🛠️ Concepts Used
- Stack (LIFO Principle)
- push()
- pop()
- peek()
- isEmpty()
- Object-Oriented Programming (OOP)

## 💻 Technologies Used
- Java
- VS Code
- Git
- GitHub

## 📂 Project Structure

```text
Browser-History-Simulator/
├── BrowserHistory.java
└── README.md
```

## ▶️ How to Run

Compile:

```bash
javac BrowserHistory.java
```

Run:

```bash
java BrowserHistory
```

## 📸 Sample Output

```text
==== Browser History ===
1. Visit Page
2. Back Page
3. Current Page
4. Show All History
5. Close Browser
```
